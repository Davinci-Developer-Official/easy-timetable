Timetable Parser & Search Tool

A simple, efficient way to transform raw timetables into searchable, structured data.

Overview

This tool is designed to parse institutional timetables (often in PDF or raw text form) into structured JSON data. Once processed, users can easily search, filter, and retrieve timetable information without scrolling endlessly through static tables.

Problem Being Solved

Students and administrators often struggle with traditional timetable formats:

  • Difficult to search for a specific unit or lecturer.
  • Time-consuming to find classes by course, year, or semester.
  • Errors when manually mapping classes from raw schedules.

This tool addresses these issues by automating the parsing and providing a searchable dataset.

How It Works

  1. Upload or parse your timetable source (e.g., PDF).
  2. The tool converts timetable entries into JSON format.
  3. Search by course abbreviation, year, semester, or day.
  4. Retrieve clean, structured results instantly.

Usage Example

Suppose you want to find all DCMS Year 3, Semester 1 units: 

{
  "course": "DCMS Y3S1",
  "day": "Monday",
  "time": "1400-1700",
  "unit": "Video Production and Editing",
  "lecturer": "Anne Mwangi",
  "venue": "L211A"
}

Key Features

  • Transforms raw timetables into clean JSON.
  • Powerful filtering by course, lecturer, year, and semester.
  • Lightweight, fast, and intuitive to use.
  • Reduces confusion and manual searching drastically.
